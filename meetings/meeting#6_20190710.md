# 미팅 기록

## DB 재밌게 쓰기

- procedure: SQL로 프로그래밍 로직 짜는게 가능
- view: 내가 필요한 데이터의 조합을 만드는게 가능
- scheduler: 타이머 같은 기능, 일정 시간에 쿼리가 동작하게 하는게 가능
- index: row count가 많이 들어갈 것 같은 테이블에 index를 걸어 select의 성능을 높임
  - DB 면접 질문에는 항상 나오는 개념
