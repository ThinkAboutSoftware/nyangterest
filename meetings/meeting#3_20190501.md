# 미팅 기록

## master branch

- 제가 보는 기준을 master branch로 할게요
- 여기에 merge를 잘 해주세요

## 문제가 생겼을 때 진행방법

- 테스트, 디버깅을 잘하자
- 테스트: 차례대로 로직의 검증
- 디버깅: 테스트를 해 봤을 때 뭔가 안되는 상황 => 디버깅 => 코드 수정 => 테스트

## 개발 환경 정리

- DB: mysql, Framework: Node, React, Language: javascript
- 실행 방법 정리: npm install, npm start
