# 회원가입완료페이지 react component로 변경하기

## flow

### before

-   클라이언트에서 회원정보입력 후 서버로 전송
-   서버에서 회원가입 성공시 사용자에게 인증메일 전송
-   사용자가 인증을 완료하면 서버에서 welcome.html 응답

### after

-   클라이언트에서 회원정보입력 후 서버로 전송
-   서버에서 회원가입 성공시 사용자에게 인증메일 전송
-   사용자가 인증을 완료하면 서버에서 프론트 url 응답

## 필요한 것

1. 회원가입완료 react component
2. 회원가입완료페이지로 연결시킬 router
